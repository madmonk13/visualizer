visualizer.py

A python script for generating music visualizer videos.


usage: visualizer.py [-h] [-o OUTPUT] [-c COVER] [-t TEXT]
                     [-p {rainbow,spring,summer,autumn,winter}] [--fps FPS]
                     [--resolution RESOLUTION] [--phone-vertical]
                     [--phone-horizontal] [--waveform-rotation {x,y,z}]
                     [--ring-rotation {x,y,z}]
                     [--starfield-rotation {none,cw,ccw}]
                     audio
